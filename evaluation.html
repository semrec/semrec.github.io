Semantic Reasoning Evaluation Challenge (SemREC)

20th International Semantic Web Conference (ISWC 2021)

For tasks 1 and 2, all our evaluations ran on a server with an AMD Ryzen Threadripper 2990WX 32-Core Processor and 128GB of RAM. The server ran on a 5.3.0-46-generic #38 18.04.1-Ubuntu SMP operating system. We use Java 1.8 and OWL API 5.1.11 in the experiments. We set the heap space to 24GB. The time-out was set to 30 minutes. We report the average time taken in 2 independent runs for each reasoning task.
